CLI 参考

快速安装（推荐）

curl -fsSL https://raw.githubusercontent.com/KevinYoung-Kw/stoi-cli/main/install.sh | bash

使用 uv（最快，自动管理 PATH）

uv tool install stoi-cli

安装完成后，运行 stoi help 验证是否可用。如果提示 command not found，请把 ~/.local/bin 加入 PATH。

CLI 命令速查

所有命令都可以通过 stoi <command> 直接调用，或进入交互式 REPL 后使用 /<command>。

命令作用
stoi report分析最新 session，输出含屎量报告与缓存命中率。
stoi report --html生成可交互的 HTML 报告并在浏览器打开。
stoi dashboard启动深度分析面板，含对话链条与逐轮 LLM 解读。
stoi start启动实时监控代理，在后台追踪当前会话的 token 效率。
stoi compare选择两个 session 做 before/after 对比，直观展示优化效果。
stoi config进入引导式配置，设置 API key、模型、语音播报等选项。
stoi help显示所有可用命令的简要说明。

REPL 命令

运行 stoi（不加参数）进入交互式 REPL，以下命令在 REPL 内使用。

命令作用
/report当前 session 的含屎量分析与输出质量简报。
/dashboard生成并打开 HTML 分析面板。
/blame粘贴 System Prompt，定位可能造成 cache miss 的字段。
/mcp一键配置 MCP，让 Claude Code 直接调用 STOI 工具。
/overview全局效率报告，汇总所有历史 session 的 token 使用情况。
/sessions切换要分析的 session（claude / opencode / gemini）。
/insights基于 LLM 生成深度优化建议（需要配置 API key）。
Ctrl + D退出 REPL。
