产品需求文档 (PRD)：STOI (Shit Token On Investment)

文档版本： v1.1
更新日期： 2026-04-07
产品定位： 开发者侧首个基于 CLI 形态的「词元 (Token) 投资回报率」分析与监控工具。

1. 产品概述
STOI 是一款旨在衡量 AI 编程工具（如 Claude Code）Token 消耗转化率的效率产品。它通过监控底层 Token 消耗，结合 LLM-as-a-judge 机制，为开发者和企业量化 “有效代码输出 (Value)” 与 “无效/冗余消耗 (Shit)” 的比例，最终给出一个直观的「含屎量」评级和改进建议。

2. 背景与痛点 (Why we build this?)
2.1 行业背景
•技术架构缺陷暴露： 随着 Claude Code 等工具源码的泄漏，开发者发现其系统提示词中硬编码了 Timestamp (时间戳)，导致 KV-Cache 大面积不命中，引发了 Token 的无谓暴涨（Token 刺客现象）。
•行业术语标准化： 国务院已正式将 Token 命名为「词元」，结束了早期名词混乱的阶段，行业进入精细化运营期。

2.2 核心痛点
•缺乏衡量标准： 目前业界只有 Token 的消耗计费，却没有衡量 Token 产出比（TOI）的指标。开发者不知道消耗的 Token 是转化成了高价值代码 (Value)，还是变成了毫无意义的废话 (Shit)。
•企业效能黑盒： 企业开始推行 Vibe Coding (全自然语言编程)，在考勤和绩效评估时，难以界定员工 AI 辅助编程的真实效率（Show me your token? Shit or value?）。
•成本失控： 无论是面向 C 端还是企业内部，缺乏行之有效的 Token 追踪机制意味着每次无效的上下文对话都在“烧钱”。

3. 核心功能需求
基于白板的右侧逻辑架构，产品核心功能模块如下：
3.1 核心指标引擎：STOI 计算
•底层数据拦截： 获取每次对话的 Context, Query 和 KV-Cache 命中状态。
•LLM as a Judge 评估模块：
￮规则校验 (Rules & Review)： 判断系统助手 (assistant_answer) 的回答是否真正解决了问题。
￮价值提取 (Outputs -> Value)： 过滤掉冗余废话，提取真正有用的代码变更或逻辑梳理。
￮计算公式化： 最终得出一个百分比或指数（STOI 指数 / 含屎量），明确指出某次交互的冗余度。

3.2 交互形态与核心命令 (CLI)
产品形态严格限定为 CLI (命令行工具)，与 Claude Code 等开发工具保持在同一工作流，不强制用户切换到 GUI 界面。

新增模块：3.3 框架审查与缓存分析 (Framework Profiling)
设计理念： 不仅要指出“含屎量”高，还要像 Git Blame 一样，指出是框架的哪一行 Prompt 生成策略导致了大量造屎 Token 浪费（Cache Miss）。
•KV-Cache 击穿检测器 (Cache-Miss Detector)：
￮拦截并对比连续两次请求的 Context 差异。
￮自动高亮（Diff）导致缓存失效的“动态元数据”（如：时间戳、无意义的随机 UUID、频繁变动的工作区绝对路径）。
•一键生成 Issue 报告：
￮当发现某个流行框架（如 Claude Code, Aider, Cline 等）存在严重的 Prompt 构造缺陷时，STOI 可以一键生成一份包含 Token 浪费测算和 Diff 证据的 Markdown 报告。
￮开发者可以直接将这份报告复制提交到对应框架的 GitHub Issues 中，倒逼官方修复。

4. 产品路线图 (Roadmap)
根据白板规划，产品将分为两个阶段进行敏捷迭代：
•Phase 1: MVP (最小可行性产品) - 验证闭环
￮分发方式： 一期支持git，后续看是否可以支持 brew 一键安装。
￮核心功能： 基础指令支持，能够集成到 Git 工作流中（git -> skill），实现最基础的 Token 消耗抓取。
•Phase 2: MVP+1 - 商业化与深度洞察
￮分发与使用： git install cli。
￮核心功能： 上线完整的 stats 数据统计和 STOI 核心算法。实现“钱花在哪里”的追踪与“改进建议”的自动生成。

5. 竞品分析与错位竞争策略
•现状： 在本次 Hackathon 大部分的项目都是 GUI、Agent，CLI 少之又少。
•破局点： 在 CLI 大行其道、Vibe Coding 兴起的时候，居然没有原生的 AI CLI 形态监控产品。STOI 采取 错位竞争 战略，直接生根在开发者的终端里，低廉的切换成本和极高的痛点契合度将是最大的护城河。
